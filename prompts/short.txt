2. ЗАКУПКА ДЛЯ СМП/СМСП — Является ли закупка для субъектов малого предпринимательства?
   Для 44-ФЗ: ищи упоминание ст.30 44-ФЗ, "закупка у субъектов малого предпринимательства", "СМП", "преимущество для СМП".
   Для 223-ФЗ: ищи "СМСП", "субъекты малого и среднего предпринимательства", "закупка среди СМСП", упоминание Постановления Правительства РФ № 1352.
   Также обращай внимание на поля извещения типа "Участниками закупки могут быть только СМП/СМСП", "Преимущества: участниками могут быть только СМСП".
   Если явно указано что закупка только для СМП/СМСП — ответь "да".
   Если явно указано что такого ограничения нет, или упоминаний вообще нет — ответь "нет".
   НЕ отвечай "информация не найдена" — это бинарный вопрос, отсутствие упоминаний = "нет".

3. НАЦИОНАЛЬНЫЙ РЕЖИМ — Как используется нацрежим?
   Ответ должен быть СТРОГО ОДНИМ из вариантов (первая строка answer):
   - "Запрет" (ПП 1875, товары из иностранных государств запрещены)
   - "Исключение из запрета" (запрет установлен, но заказчик указал основание исключения)
   - "Ограничение" (ПП 1875, ограничение для иностранных товаров при наличии российских предложений)
   - "Преимущество" (ПП 1875 или ПП 925, преимущество российским товарам)
   - "Не предусмотрено" (нацрежим не применяется)

   ВАЖНО: Преимущество для СМП по ст.30 44-ФЗ — это НЕ нацрежим. Нацрежим — только про страну происхождения товара (ПП 1875, ПП 925).

   ОСНОВАНИЯ ДЛЯ ИСКЛЮЧЕНИЯ ИЗ ЗАПРЕТА (ищи эти формулировки, если есть запрет):
   - "сумма закупки до 1 млн рублей" / "НМЦК менее 1 млн"
   - "получено разрешение Минпромторга"
   - "отсутствие российского аналога" / "товар не производится в РФ"

   После основного варианта (в answer) — дай подробности: какой ОКПД2, какое основание исключения если есть.
